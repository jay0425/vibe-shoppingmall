# fe 환경 설정 및 UI 구현

1. 첨부한 컨벤션 참고해서 /client 폴더에 next.js, tailwindCSS, ts, tanstack-query,pnpm 조합으로 프로젝트 환경 셋팅 해줘.
2. /client 폴더 안에 app, components, lib, public 폴더들 다른 곳에서 복붙해왔어. 첨부한 컨벤션 내용 참고해서 여기 프로젝트에 맞게 수정해줘.

# 기능 연결

1. 루트 리드미 컨벤션 참고해서 작성해줘.

/client 폴더에서 FSD 폴더구조가 되게 해줘.

첨부한 컨트롤러를 참고해서
client/src/app/signup/page.tsx
에서 버튼 클릭 시 회원가입 되도록 로직 연결해줘.

/server 아래 파일들은 참고만 하고 건들지 말아줘.

그리고 만약에 API와 UI가 안맞는 부분이 있으면 나중에 보고 주고 일단 API 연결부터 해줘

2.  이메일 형식 검사,
    중복 가입 확인,
    비밀번호는
    8자 이상 20자 이하
    영문 + 숫자 + 특수문자 포함
    공백 불가
    허용된 특수문자 외 입력 차단
    password, 반복 문자, 단순 숫자 나열 등 차단

이름·이메일·비밀번호는 필수값,

모든 입력값은 공백 제거, 특수문자/이모지 불가

---

위의 내용을 반영해서 정규식 및 유효성 검사 로직 추가해줘.

3. 뒤로 가기 후 다시 가입 시도 시
   →회원가입폼이 초기화되어 입력하던 유저정보는 전부 리셋되어 있어야 함.
   가입 중에 인터넷 네트워크가 끊겨버림
   → 에러 상황에 따른 알맞은 유저가 이해할 수 있는 메세지 출력
